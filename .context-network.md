# Project Context Network

## Location
The context network for this project is located at: `context/`

## Purpose
This context network contains all planning documents, story development notes, architectural decisions, and team coordination information for The Kepler Testimonies collaborative fiction project.

## Usage Guidelines
- ALL planning documents, story outlines, and design discussions MUST be stored within the context network
- Do NOT place planning or architecture documents in the project root
- Project files outside the context network should only be:
  - Final manuscript files
  - Published/exported content
  - Build/automation scripts
